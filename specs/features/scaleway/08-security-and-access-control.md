# Scaleway Stories - Feature 08 Security and Access Control

Feature source: [08-security-and-access-control.md](/Users/kees/data/projects/archive/features/08-security-and-access-control.md)

## Scaleway components

- IAM
- Key Manager
- Secret Manager
- API authorization layer
- bucket policies

## Stories

- `SCW-08-01` Authentication integration for workforce users and workloads.
- `SCW-08-02` Role-based authorization for finance, legal, audit and operations.
- `SCW-08-03` Scope-based authorization by legal entity, document type and confidentiality class.
- `SCW-08-04` Encryption in transit and managed-key usage for storage and APIs.
- `SCW-08-05` Fine-grained rights for read, export, metadata update, disposition and legal hold.
- `SCW-08-06` Extra protection paths for sensitive records and high-impact admin actions.

## Acceptance focus

- authorization decisions are explainable
- encryption is baseline, not the only control
- high-risk actions have stronger checks
